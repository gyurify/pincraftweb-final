Features

Built with the full workflow needed for accurate button pin production.

Pincraft combines live editing, layer control, design storage, print preparation, PDF export, and real-size preview tools in one focused workspace.

Interactive Button Editor

Upload and position images, add draggable and resizable text layers, and apply rotation, scaling, transparency, and stretch adjustments with real-size preview support.

Layer Management System

Show or hide layers, reorder elements, perform undo and redo actions, and drag or resize objects within the design canvas.

Multiple Design Storage

Save multiple button designs locally and access them through a gallery interface for editing or reuse.

Smart Print Layout Generator

Automatically arranges multiple circular button designs on an A4 sheet for an optimized printing layout.

PDF Export

Generate professional and print-ready PDF files directly from the application.

Live Preview and Real Measurement

Ensures accurate scaling based on actual button size standards to help minimize printing errors.

About

Made to simplify the full button pin layout process.

Pincraft is built to help users manage, design, preview, and print button pin layouts easily through one clear and organized workflow.

Pincraft brings the main stages of button pin production into one focused workspace. Instead of jumping between separate steps, users can manage projects, build layouts, review previews, and prepare output with a smoother process from start to finish.

The goal is to make button pin layout work easier to understand and more efficient to complete, while keeping the interface clean, practical, and ready for real production use.
